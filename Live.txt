파이썬으로, 인터넷에 있는 날씨 정보를 가져와, 내가 원하는 정보만 출력.

날씨 데이터는 직접 모으기 어렵기 때문에 인터넷에 있는 데이터를 가져오면 된다.

어떻게? 우리(클라이언트)가 네이버(서버)에 정보를 요청하면, 서버가 정보를 클라이언트에게 돌려준다.

그럼 어떻게 정보를 요청하나? 웹 브라우저에 url을 입력하거나, 서버에 정보를 요청하는 파이썬 코드를 작성한다.

파이썬 코드 어떻게 작성하지? vscode의 terminal을 열어서 requests 라이브러리를 설치. $ pip install requests

API: 클라이언트가 원하는 기능을 수행하기 위해 서버 측에서 만들어 놓은 프로그램.
서버 측에서 '특정 주소로 요청이 오면 정해진 기능을 수행하는' API를 미리 만들어 둠.

정보를 수집하기 위해서는 두 가지를 알아야 한다.
1. 정보를 가지고 있는 서버
2. 해당 서버가 제공하는 API

즉, 우리는 정확히는 서버에 요청하는 게 아니라, 서버가 만들어둔 API에 요청하는 것.

TMDB: 영화 정보 API.

JSON: API가 사용하는 데이터 형식. list 안의 dict처럼 생겼음. 이걸 실제로 list 안의 dict 형태로 받는 method가 json().

json()으로 데이터를 의미있는 구조로 분석하고 해석하는 과정을 파싱 parsing이라고 한다.

json.loads(data): JSON 형식의 문자열을 parsing하여 dict로 변환.
